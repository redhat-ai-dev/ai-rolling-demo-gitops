## End-to-end test automation 

* Automated end-to-end scenarios are developed with the [Playwright](https://playwright.dev/) framework.  
* Review the [Developer Hub end-to-end Test](https://docs.google.com/document/d/18zz2FAb0p0luqrIlzL3UK6A86pxojRp1X4xG-boZvyI/edit#heading=h.10lwnlf1c4wb) document to get familiar with the test automation framework, the test automation code repository, how to add new tests, how to execute and restart tests, and required details to develop end-to-end scenarios.
